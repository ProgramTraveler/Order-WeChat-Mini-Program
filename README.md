## 使用注意事项

* 本小程序是基于微信云开发
* 个人商户无法开启支付功能
* 所以实现的是到付

## 开发注意事项
* 请在`project.config.json`中 修改你自己的 appid
```exp
  "appid": ""
```
* 在`project.config.json`中 配置自己的云函数根目录
```exp
"cloudfunctionRoot": "cloud/",
```
* 在`app.js`中配置自己的云开发环境
```exp
    wx.cloud.init({
      env:"自己的云开发环境id",
      traceUser:true
    });
```
